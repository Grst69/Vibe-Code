# ⚡ Vibe Code

> Ship with AI, Not Against It.

The next generation AI-powered coding agent. Describe what you want in plain English, watch it build. From idea to production in minutes, not months.

## 🚀 Live Demo

**[https://grst69.github.io/Vibe-Code/](https://grst69.github.io/Vibe-Code/)**

## ✨ Features

- 🧠 **Multi-Model Brain** — Switch between Claude, GPT-5, Gemini, and open-source models
- ⚡ **Instant Preview** — See your app live as it's being built
- 🔗 **MCP Integration** — Connect to any MCP server
- 🛡️ **Auto Security** — Built-in security scanning
- 📦 **One-Click Deploy** — Vercel, Netlify, Cloudflare Workers, AWS
- 🎨 **Design to Code** — Paste a screenshot and get production-ready UI

## 🛠️ Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- Particle system with canvas
- CSS animations & transitions
- Intersection Observer for scroll effects

## 📦 How to Use

Just open `index.html` in your browser or visit the [live demo](https://grst69.github.io/Vibe-Code/).
